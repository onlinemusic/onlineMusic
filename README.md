# onlineMusic

在线音乐平台


项目总监要求:
--------------------------
-   每个人fork到本地仓库，需要合并再与主项目合并。
-   目前打算采用的spring、springMVC、Hibernate  ssh框架。
-   目前开发工具为idea
<br>

如何与团队项目保持同步(重要)
--------------------------

-   第一次时需要,与团队仓库建立联系

``` {.bash}
git remote add upstream https://github.com/onlinemusic/onlineMusic.git
```

-   工作前后要运行这几条命令,和团队项目保持同步

``` {.bash}
git fetch upstream
git merge upstream/master
```
